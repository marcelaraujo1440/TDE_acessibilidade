Ferramentas e Tecnologias

1. WAVE Accessibility Tool

Uma extensão de navegador que avalia automaticamente a acessibilidade do site, destacando problemas e fornecendo sugestões de correção.Saiba mais

2. NVDA (NonVisual Desktop Access)

Um leitor de tela gratuito e amplamente usado por pessoas com deficiência visual para navegar na web. Saiba mais

3. Axe DevTools

O Axe DevTools é uma ferramenta para desenvolvedores que verifica automaticamente a acessibilidade de sites e aplicativos. Ela identifica problemas de acessibilidade, oferece orientações para corrigi-los e ajuda a criar experiências mais inclusivas para todos os usuários Saiba mais

4. Color Contrast Checker

O Color Contrast Checker é uma ferramenta que verifica se a combinação de cores de texto e fundo atende aos requisitos de contraste necessários para garantir boa legibilidade, especialmente para pessoas com deficiências visuais, como daltonismo. Saiba mais

5. Hand Talk

A Hand Talk é uma plataforma que traduz simultaneamente conteúdos em português para a Língua Brasileira de Sinais (Libras), visando a inclusão social de pessoas surdas Saiba mais
